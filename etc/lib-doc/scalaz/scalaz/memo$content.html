Scalaz 2.3 API Specification

This document is the API specification for Scalaz 2.3

Class Summary
trait ImmutableAssociation [K, V, T] extends AnyRef
Represents any association from a K (key) to V (value) over the immutable type T, which represents the association.
trait Memo [K, V] extends AnyRef
An abstraction representing memoisation used in many dynamic programming (DPA) algorithms and code that makes efforts to follow a purely functional style. Loosely inspired by Stretching the storage manager: weak pointers and stable names in Haskell (Simon Peyton Jones, Simon Marlow, and Conal Elliott, IFL'99.) Further abstractions, such as purging the memoisation table, are achieved by abstracting over the type that maintains the assocation (and so, may define means of purging themselves).
trait MutableAssociation [K, V, T] extends AnyRef
Represents any association from a K (key) to V (value) over the mutable type T, which represents the association.
trait SizedMemo [K] extends AnyRef
Memoisation within a given range or size.
trait TMemo [K, V, T] extends AnyRef
An association from some type to a memoisation technique backed by that type.
